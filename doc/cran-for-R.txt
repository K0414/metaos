Recommended packages for R:


 1.- FORECAST
    Forecast http://cran.r-project.org/web/packages/forecast/index.html
        > install.packages("forecast")

    In MacOSX, use command line version of R, or these commands:
        Download the package 
        sudo /usr/bin/R CMD INSTALL  thepackage

        See http://www.sysarchitects.com/node/94

    Then, to use the library, in R:
        > library(forecast)
